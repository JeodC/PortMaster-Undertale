## Installation
Download the Windows version from Steam and put all gamedata in `ports/undertale/gamedata`.

## Default Gameplay Controls
| Button | Action |
|--|--|
|START|Confirm|
|D-PAD|Move|
|LEFT ANALOG|Move|
|A|Confirm|
|B|Cancel|
|X|In-Game Menu|
|Y|In-Game Menu|
